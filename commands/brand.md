description: Strip AI design tells and add intentional character
---

Run the design-for-ai skill focused on design identity and differentiation.

1. Read references/ai-tells.md to understand common AI design fingerprints.
2. Scan the current design/code for AI tells (generic fonts, cyan-on-dark, card-everything, glassmorphism, bounce easing, gradient text on metrics, dark mode with glowing accents).
3. For each tell found:
   - Identify it specifically
   - Explain WHY it reads as generic (grounded in design theory — Ch 1 on authentic style, Ch 4 on style from tech+culture)
   - Propose an intentional alternative that serves the design's purpose
4. Ask: "What should make this design MEMORABLE? What's the one thing someone remembers?"
5. Apply changes that add character without adding complexity.

The goal isn't "different for different's sake." It's design decisions that trace back to purpose. Ch 1: "Design is the fundamental soul of a human-made creation." If you can't explain WHY a visual choice was made, it's decoration, not design.
