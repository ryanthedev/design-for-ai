description: Apply visual design principles — review, create, or get specific help
---

Invoke the `design-for-ai` skill. This command supports two core modes:

**CHECKER mode** (default when reviewing existing work): Run a systematic design audit covering purpose, typography, proportions, composition, visual hierarchy, color, motion, interaction, responsive design, and design identity.

**APPLIER mode** (when creating from scratch): Walk through a phased design process — foundation, structure, typography, composition & hierarchy, color, motion & interaction, responsive, and technical implementation.

**For targeted help, suggest these commands:**
- `/design` — establish design foundations (purpose, audience, direction)
- `/exam` — detailed design audit with theory-backed diagnosis
- `/brand` — strip AI tells, add intentional character
- `/fonts` — typography selection and configuration
- `/color` — build a color system
- `/flow` — motion, interaction states, responsive behavior
- `/hone` — final quality pass

If the user provided arguments, use them as context. If no arguments, ask what they'd like help with.
