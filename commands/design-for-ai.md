---
description: Apply visual design principles — review a design or guide creation from scratch
---

Invoke the `design-for-ai` skill to apply visual design principles. This command supports two modes:

**CHECKER mode** (default when reviewing existing work): Run a systematic design audit covering purpose, typography, proportions, composition, visual hierarchy, color, and SEO.

**APPLIER mode** (when creating from scratch): Walk through a phased design process — foundation, structure, typography, composition & hierarchy, color, and technical implementation.

If the user provided arguments with this command, use them as context for what to review or create. If no arguments were provided, ask what they'd like help with.
