Run the design-for-ai skill focused on typography decisions.

1. Read references/chapter-03-typography.md and references/appendix-fonts-and-typography.md.
2. Use the "I Need to Pick Fonts" decision tree from references/checklists.md.
3. Walk through:
   - **Medium**: Screen or print? What resolution/devices?
   - **Body font**: Select based on medium-form relationship (Ch 3)
   - **The "n" test**: Identify letter structure (humanist, geometric, realist)
   - **Pairing** (if needed): Match structures for harmony or use extreme contrast
   - **Scale**: Build a proportional type scale from references/chapter-05-proportions.md (e.g., 3:4 ratio)
   - **Leading**: Set 1.2-1.4em for body
   - **Characters**: Smart quotes, proper dashes, no fake bold/italic
4. Output: recommended font stack, type scale values, CSS custom properties, and rationale for each choice grounded in the book's principles.

If the user provided arguments, use them as context. If not, ask about the target medium and desired mood.
